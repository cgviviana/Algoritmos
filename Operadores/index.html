Qué es un Operador
El simbolo que indica una operación
Un operador es un símbolo que le dice al algoritmo que operación realizar entre uno o dos valores. El resultado puede ser un número, un texto o un valor lógico (Verdadero/Falso).
Operadores Aritméticos
Para calcular con números
Producen un resultado numérico. Se usan con variables de tipo Entero o Real. El operador especial es MOD: devuelve el RESTO de una division entera.
SimboloNombreEjemploResultado
+Suma8 + 311
-Resta8 - 35
*Multiplicacion4 * 312
/Division10 / 42.5
MODResto / Modulo10 MOD 31
^Potencia2 ^ 8256
Operadores Relacionales
Para comparar dos valores
Comparan dos valores y devuelven SIEMPRE un resultado Lógico (Verdadero o Falso). Se usan en estructuras algorítimicas avanzadas de Si, Mientras y Repetir para tomar decisiones.
SimboloSignificadoEjemploResultado
>Mayor que8 > 3Verdadero
<Menor que2 < 1Falso
>=Mayor o igual5 >= 5Verdadero
<=Menor o igual3 <= 2Falso
=Igual a4 = 4Verdadero
<>Distinto de5 <> 3Verdadero
Operadores Lógicos
Para combinar condiciones
Se usan cuando necesitas verificar dos o más condiciones al mismo tiempo (Y), al menos una (O) o negar una condicion (NO).
SimboloNombreCuándo da Verdadero
YY lógico (AND)Cuando AMBAS condiciones son Verdaderas
OO lógico (OR)Cuando AL MENOS UNA es Verdadera
NONegacion (NOT)Invierte: si era Verdadero → Falso y viceversa
Operador de Asignacion
La flecha <- que guarda un valor
El operador <- (flecha) asigna el resultado del lado derecho a la variable del lado izquierdo. Es diferente de = (comparacion). El lado derecho se calcula primero.
Un error comun: confundir <- (asignar) con = (comparar). En PSeInt, la flecha <- siempre asigna. El signo = solo compara dentro de condiciones Si / Mientras.
// Asignar un valor directo
puntaje <- 0
// Asignar el resultado de una expresion
total <- precio * cantidad
// Acumular (leer el valor actual y sumarle algo)
puntaje <- puntaje + 100
// Asignar resultado de comparacion
esMayor <- edad >= 18
Concatenacion de Texto
Unir cadenas de Caracter con +
En PSeInt se usa el simbolo + para unir (concatenar) dos o mas textos de tipo Caracter. El resultado es un nuevo texto que los contiene a todos juntos.
La concatenacion solo funciona entre Caracter. Si mezclas un número con texto, PSeInt puede dar error o convertir automaticamente. Es buena practica convertir el número a texto antes.
// Unir dos textos
saludo <- "Hola " + nombre
// Resultado: "Hola Sofia" (si nombre = "Sofia")

// Incluir un número en un mensaje

Precedencia de Operadores
Quien se calcula primero
Cuando hay varios operadores en una expresion, PSeInt los calcula en un orden fijo. Los parentesis siempre van primero. Si tienes dudas, usa parentesis para forzar el orden que necesitas.
#OperadoresDescripcion
1( )Parentesis — siempre primero
2^ NOPotencia y negacion logica
3* / MODMultiplicacion, division, modulo
4+ -Suma y resta
5> < >= <= = <>Relacionales
6Y OLógicos — ultimo
// Sin parentesis: * va antes que +
resultado <- 2 + 3 * 4 // = 14 (no 20)
// Con parentesis: controlas el orden
resultado <- (2 + 3) * 4 // = 20

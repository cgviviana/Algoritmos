¿Qué es un dato de texto?
Un dato de texto es cualquier información formada por letras, palabras, frases o símbolos que queremos guardar como mensaje, no como número.
Cuando un programa guarda un nombre, una dirección, un mensaje de chat o un código como "ABC-01", está usando un dato de tipo texto. No sirve para hacer sumas, solo para leer y mostrar información.
Nombre Dirección Mensaje Código Color
Ejemplos del mundo real: el nombre de un estudiante, la dirección de una tienda, el color favorito de un usuario, un número de placa como "ABC-123".
String — Cadena de texto
También llamada: char · varchar · texto · str
Regla: siempre va entre comillas " " o ' '. Si tiene comillas es String, sin importar qué haya adentro.
Así se escribe un dato de texto en código. Las comillas le dicen al computador: "esto es texto, no lo calcules". Incluso un número entre comillas se convierte en texto.
"Hola""Colombia"'Gato'"abc123""2024""true"
nombre = "Sofía"
ciudad = 'Bogotá'
placa = "ABC-123"
num_texto = "42" ← texto, ¡no número!
Integer — Número entero
También: int · long · short · byte
Regla: número completo, sin punto decimal y sin comillas. Puede ser positivo, negativo o cero.
Se usa cuando necesitamos contar cosas exactas: edades, posiciones, cantidades, años. No tiene parte decimal porque representa unidades completas.
5-31000-9992024
edad = 25
temperatura = -10
puntaje = 500
año = 2024
Float / Double — Número real
También llamado: float · double · decimal · real
Regla: tiene un punto (.) decimal y va sin comillas. ¡Ojo! 1.0 también es float aunque el decimal sea cero.
Se usa para medidas, precios, promedios y cualquier valor que necesite decimales. Double tiene más precisión que float (más decimales exactos).
3.14-0.59.991.0100.75-2.8
precio = 19.99
promedio = 4.5
pi = 3.14159
altura = 1.75
Boolean — Booleano
También llamado: bool · boolean · lógico · bit
Regla: SOLO puede ser true (verdadero) o false (falso). Sin comillas, sin números. Solo esas dos palabras.
Como un interruptor de luz: solo tiene dos estados. Se usa para condiciones, permisos y estados como activo/inactivo, sí/no, encendido/apagado.
truefalse
esMayor = true
tieneDescuento = false
estaActivo = true
"true" con comillas → es String, ¡no Boolean!
